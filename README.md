## ğŸš€ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sivchheng16/typing-math-game.git
   cd typing-math-game
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
- Run the **Web** version:
  ```bash
  npm run dev
  ```
- Run the **Electron** desktop version:
  ```bash
  npm run electron:dev
  ```

### Building
- **Build for Web**:
  ```bash
  npm run build
  ```
- **Package for Desktop (Linux/AppImage)**:
  ```bash
  npm run electron:build
  ```
